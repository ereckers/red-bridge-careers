== Installation ==

1. Simply install through the WordPress Plugins administrative screen

Manual Installation

1. Download the plugin and unzip it.
2. Upload the folder /red-bridge-careers to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Installation finished.
